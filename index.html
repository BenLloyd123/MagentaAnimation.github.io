What is Number Run?

Number Run is a fun and interactive maths game for students between the ages of 4 and 18.

Number Run gives a fun platform for children to learn, have fun and compete with friends. With customisation options from reception all the way up to A level. This is perfect for children who want to combine fun with mathematics.
